# Pin Input

A PIN / OTP input built on [Ark UI Pin Input](https://ark-ui.com/docs/components/pin-input). The convenience component renders `length` cells, a label, and `PinInputHiddenInput` for native form submission. Use the compound parts to customize each segment.

<Preview>
  <PinInput label="Verification code" length={6} />

```tsx title="pin-input-demo.tsx" showLineNumbers
import { PinInput } from "@/components/ui/pin-input";

export default function Example() {
  return <PinInput label="Verification code" length={6} />;
}
```
</Preview>

## Installation

```bash
liminal add pin-input
```

## Usage

`length` sets how many segments are rendered (default `4`). Additional props are forwarded to `PinInputRoot` (for example `otp`, `type`, `value`, `onValueChange`).

<Preview>
  <PinInput label="Short code" length={4} />

```tsx title="pin-input-usage.tsx" showLineNumbers
import { PinInput } from "@/components/ui/pin-input";

export default function Example() {
  return <PinInput label="Short code" length={4} />;
}
```

</Preview>

## Examples

### Compound API

Exports: `PinInputRoot`, `PinInputLabel`, `PinInputControl`, `PinInputField`, `PinInputHiddenInput`. Render one `PinInputField` per index when composing manually.

```tsx title="pin-input-compound.tsx" showLineNumbers
import {
  PinInputRoot,
  PinInputLabel,
  PinInputControl,
  PinInputField,
  PinInputHiddenInput,
} from "@/components/ui/pin-input";

export default function Example() {
  return (
    <PinInputRoot count={4}>
      <PinInputLabel>Code</PinInputLabel>
      <PinInputControl>
        <PinInputField index={0} />
        <PinInputField index={1} />
        <PinInputField index={2} />
        <PinInputField index={3} />
      </PinInputControl>
      <PinInputHiddenInput />
    </PinInputRoot>
  );
}
```
