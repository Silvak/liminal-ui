# Tooltip

A tooltip component built with Ark UI for showing additional information on hover or focus. Use `TooltipSimple` for the common case and the composed API for full control.

<Preview>
  <TooltipSimple content="Additional information about this action.">
    <Button variant="outline">Hover me</Button>
  </TooltipSimple>

```tsx title="tooltip-quick.tsx" showLineNumbers
import { TooltipSimple } from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <TooltipSimple content="Additional information about this action.">
      <Button variant="outline">Hover me</Button>
    </TooltipSimple>
  );
}
```

</Preview>

## Installation

```bash
liminal add tooltip
```

## Usage

### Quick API (recommended)

<Preview>
  <TooltipSimple content="Additional information about this action.">
    <Button variant="outline">Hover me</Button>
  </TooltipSimple>

```tsx title="tooltip-quick.tsx" showLineNumbers
import { TooltipSimple } from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <TooltipSimple content="Additional information about this action.">
      <Button variant="outline">Hover me</Button>
    </TooltipSimple>
  );
}
```

</Preview>

### Composed API (full control)

<Preview>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="outline">Hover me</Button>
    </TooltipTrigger>
    <TooltipPositioner>
      <TooltipContent>
        Additional information about this action.
      </TooltipContent>
    </TooltipPositioner>
  </Tooltip>

```tsx title="tooltip-composed.tsx" {3,4,5,6,7,8,9,10} showLineNumbers
import {
  Tooltip,
  TooltipTrigger,
  TooltipPositioner,
  TooltipContent,
} from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <Tooltip>
      <TooltipTrigger asChild>
        <Button variant="outline">Hover me</Button>
      </TooltipTrigger>
      <TooltipPositioner>
        <TooltipContent>
          Additional information about this action.
        </TooltipContent>
      </TooltipPositioner>
    </Tooltip>
  );
}
```

</Preview>

## Examples

### With link

Wrap any focusable element with `TooltipTrigger asChild`; the tooltip appears on hover or focus.

<Preview>
  <Tooltip>
    <TooltipTrigger asChild>
      <a href="#" className="text-sm underline">Learn more</a>
    </TooltipTrigger>
    <TooltipPositioner>
      <TooltipContent>Opens the documentation.</TooltipContent>
    </TooltipPositioner>
  </Tooltip>

```tsx title="tooltip-link.tsx" showLineNumbers
<Tooltip>
  <TooltipTrigger asChild>
    <a href="#">Learn more</a>
  </TooltipTrigger>
  <TooltipPositioner>
    <TooltipContent>Opens the documentation.</TooltipContent>
  </TooltipPositioner>
</Tooltip>
```

</Preview>
