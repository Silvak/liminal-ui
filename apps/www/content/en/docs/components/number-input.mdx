# Number Input

`NumberInput` adds step controls and keyboard-friendly numeric editing.

<Preview>
  <NumberInput label="Quantity" defaultValue="2" min={1} max={10} />

```tsx title="number-input-demo.tsx" showLineNumbers
import { NumberInput } from "@/components/ui/number-input";

export default function Example() {
  return <NumberInput label="Quantity" defaultValue="2" min={1} max={10} />;
}
```
</Preview>

```bash
liminal add number-input
```
