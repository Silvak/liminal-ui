# Input

A text input component with focus styles and support for error states via `aria-invalid`.

<Preview>
  <Input placeholder="Email" type="email" />

```tsx title="input-basic.tsx" {3} showLineNumbers
import { Input } from "@/components/ui/input";

export default function Example() {
  return <Input placeholder="Email" type="email" />;
}
```

</Preview>

## Examples

### Error state

Use `aria-invalid="true"` to show error styling. Pair with `Label` and error text for accessibility.

<Preview>
  <Input placeholder="Email" type="email" aria-invalid="true" />

```tsx title="input-error.tsx" {3} showLineNumbers
import { Input } from "@/components/ui/input";

export default function Example() {
  return (
    <Input
      placeholder="Email"
      type="email"
      aria-invalid="true"
    />
  );
}
```

</Preview>
