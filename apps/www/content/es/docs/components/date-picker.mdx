# Date Picker

`DatePicker` ofrece un control listo para usar con input, trigger y contenido componible.

<Preview>
  <DatePicker label="Fecha" placeholder="Selecciona una fecha" />

```tsx title="date-picker-demo.tsx" showLineNumbers
import { DatePicker } from "@/components/ui/date-picker";

export default function Example() {
  return <DatePicker label="Fecha" placeholder="Selecciona una fecha" />;
}
```
</Preview>

## Instalación

```bash
liminal add date-picker
```
