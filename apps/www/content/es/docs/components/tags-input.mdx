# Tags Input

`TagsInput` permite crear chips/etiquetas con teclado y quitarlas en línea.

<Preview>
  <TagsInput label="Etiquetas" placeholder="Escribe y presiona Enter..." />

```tsx title="tags-input-demo.tsx" showLineNumbers
import { TagsInput } from "@/components/ui/tags-input";

export default function Example() {
  return <TagsInput label="Etiquetas" placeholder="Escribe y presiona Enter..." />;
}
```
</Preview>

## Instalación

```bash
liminal add tags-input
```
