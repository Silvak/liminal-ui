# Badge

Componente badge para mostrar estados, categorías o conteos. Úsalo para etiquetas, rótulos o indicadores numéricos pequeños.

<Preview>

```tsx title="badge-basic.tsx" {3} showLineNumbers

```

</Preview>

```bash

```
