# Field

`Field` estructura entradas accesibles con etiqueta, texto de ayuda y mensaje de error.

<Preview>
  <Field
    label="Email"
    helperText="Solo lo usaremos para notificaciones de cuenta."
    inputProps={{ placeholder: "tu@ejemplo.com", type: "email" }}
  />

```tsx title="field-demo.tsx" showLineNumbers
import { Field } from "@/components/ui/field";

export default function Example() {
  return (
    <Field
      label="Email"
      helperText="Solo lo usaremos para notificaciones de cuenta."
      inputProps={{ placeholder: "tu@ejemplo.com", type: "email" }}
    />
  );
}
```
</Preview>

```bash
liminal add field
```
