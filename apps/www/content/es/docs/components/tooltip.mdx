# Tooltip

Componente tooltip construido con Ark UI para mostrar información adicional al pasar el ratón o al enfocar. Usa `TooltipSimple` para el caso común y la API compuesta para control total.

<Preview>
  <TooltipSimple content="Información adicional sobre esta acción.">
    <Button variant="outline">Pasa el ratón</Button>
  </TooltipSimple>

```tsx title="tooltip-rapido.tsx" showLineNumbers
import { TooltipSimple } from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <TooltipSimple content="Información adicional sobre esta acción.">
      <Button variant="outline">Pasa el ratón</Button>
    </TooltipSimple>
  );
}
```

</Preview>

## Instalación

```bash
liminal add tooltip
```

## Uso

### API rápida (recomendada)

  <TooltipSimple content="Información adicional sobre esta acción.">
    <Button variant="outline">Pasa el ratón</Button>
  </TooltipSimple>

```tsx title="tooltip-rapido.tsx" showLineNumbers
import { TooltipSimple } from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <TooltipSimple content="Información adicional sobre esta acción.">
      <Button variant="outline">Pasa el ratón</Button>
    </TooltipSimple>
  );
}
```

</Preview>

### API compuesta (control total)

<Preview>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="outline">Pasa el ratón</Button>
    </TooltipTrigger>
    <TooltipPositioner>
      <TooltipContent>
        Información adicional sobre esta acción.
      </TooltipContent>
    </TooltipPositioner>
  </Tooltip>

```tsx title="tooltip-compuesto.tsx" {3,4,5,6,7,8,9,10} showLineNumbers
import {
  Tooltip,
  TooltipTrigger,
  TooltipPositioner,
  TooltipContent,
} from "@/components/ui/tooltip";
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <Tooltip>
      <TooltipTrigger asChild>
        <Button variant="outline">Pasa el ratón</Button>
      </TooltipTrigger>
      <TooltipPositioner>
        <TooltipContent>
          Información adicional sobre esta acción.
        </TooltipContent>
      </TooltipPositioner>
    </Tooltip>
  );
}
```

</Preview>

## Ejemplos

### Con enlace

Envuelve cualquier elemento enfocable con TooltipTrigger asChild; el tooltip aparece al pasar el ratón o al enfocar.

<Preview>
  <Tooltip>
    <TooltipTrigger asChild>
      <a href="#" className="text-sm underline">Saber más</a>
    </TooltipTrigger>
    <TooltipPositioner>
      <TooltipContent>Abre la documentación.</TooltipContent>
    </TooltipPositioner>
  </Tooltip>

```tsx title="tooltip-link.tsx" showLineNumbers
<Tooltip>
  <TooltipTrigger asChild>
    <a href="#">Saber más</a>
  </TooltipTrigger>
  <TooltipPositioner>
    <TooltipContent>Abre la documentación.</TooltipContent>
  </TooltipPositioner>
</Tooltip>
```

</Preview>
