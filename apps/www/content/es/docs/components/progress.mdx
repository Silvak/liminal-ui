# Progress

Barra de progreso basada en [Ark UI Progress](https://ark-ui.com/docs/components/progress). Usa la API de conveniencia con `value` (y opcional `showLabel`) o la API compuesta para un layout personalizado.

<Preview>
  <Progress value={60} showLabel />

```tsx title="progress-convenience.tsx" showLineNumbers
import { Progress } from "@/components/ui/progress";

export default function Example() {
  return <Progress value={60} showLabel />;
}
```

</Preview>

## Instalación

```bash
liminal add progress
```

## Uso

Pasa `value` (0–100 por defecto) y, si quieres, `min`, `max` y `showLabel` para mostrar el porcentaje sobre la pista.

<Preview>
  <div className="w-full space-y-4">
    <Progress value={33} showLabel />
    <Progress value={80} showLabel />
  </div>

```tsx title="progress-usage.tsx" showLineNumbers
import { Progress } from "@/components/ui/progress";

<Progress value={33} showLabel />
<Progress value={80} showLabel />
```

</Preview>

### API compuesta

Usa `ProgressRoot`, `ProgressLabel`, `ProgressValueText`, `ProgressTrack` y `ProgressRange` para un layout a medida.

```tsx title="progress-compound.tsx" showLineNumbers
import {
  ProgressRoot,
  ProgressLabel,
  ProgressValueText,
  ProgressTrack,
  ProgressRange,
} from "@/components/ui/progress";

export default function Example() {
  return (
    <ProgressRoot value={50} min={0} max={100}>
      <div className="mb-2 flex justify-between">
        <ProgressLabel>Subida</ProgressLabel>
        <ProgressValueText />
      </div>
      <ProgressTrack>
        <ProgressRange />
      </ProgressTrack>
    </ProgressRoot>
  );
}
```
