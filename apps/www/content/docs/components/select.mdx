# Select

A select component built with Ark UI. Use the convenience API with an `items` array for most cases, or the compound API for custom markup and grouping.

<Preview>
  <Select
    items={[
      { label: "React", value: "react" },
      { label: "Vue", value: "vue" },
      { label: "Svelte", value: "svelte" },
    ]}
    placeholder="Select a framework"
  />

```tsx title="select-convenience.tsx" {3,4,5,6,7,8,9,10} showLineNumbers
import { Select } from "@/components/ui/select";

const items = [
  { label: "React", value: "react" },
  { label: "Vue", value: "vue" },
  { label: "Svelte", value: "svelte" },
];

export default function Example() {
  return <Select items={items} placeholder="Select a framework" />;
}
```

</Preview>

### Compound API

For grouped options or custom styling, use `SelectRoot`, `SelectControl`, `SelectTrigger`, `SelectValue`, `SelectContent`, `SelectGroup`, and `SelectItem`. You must create a collection with `ArkSelect.createCollection` (or use the convenience component internally).

<Preview>
  <Select
    items={[
      { label: "React", value: "react" },
      { label: "Vue", value: "vue" },
      { label: "Svelte", value: "svelte" },
    ]}
    placeholder="Select a framework"
  />
</Preview>

```tsx title="select-compound.tsx" showLineNumbers
import {
  SelectRoot,
  SelectControl,
  SelectTrigger,
  SelectValue,
  SelectContent,
  SelectGroup,
  SelectItem,
} from "@/components/ui/select";

const items = [
  { label: "React", value: "react" },
  { label: "Vue", value: "vue" },
  { label: "Svelte", value: "svelte" },
];

export default function Example() {
  return (
    <SelectRoot
      collection={ArkSelect.createCollection({
        items,
        itemToString: (item) => item.label,
        itemToValue: (item) => item.value,
      })}
    >
      <SelectControl>
        <SelectTrigger>
          <SelectValue placeholder="Select a framework" />
        </SelectTrigger>
      </SelectControl>
      <SelectContent>
        <SelectGroup>
          {items.map((item) => (
            <SelectItem key={item.value} item={item}>
              {item.label}
            </SelectItem>
          ))}
        </SelectGroup>
      </SelectContent>
    </SelectRoot>
  );
}
```

For most use cases, the convenience `<Select items={...} />` is enough; use the compound API when you need groups or custom item rendering.
