# Button

A button component built with Ark UI that supports multiple variants and sizes.

<Callout type="tip" title="When to use Button">
  Use `variant="default"` for primary actions and `variant="outline"` for secondary
  actions. Keeping this hierarchy improves UI readability.
</Callout>

## Installation

```bash
liminal add button
```

## Usage

<Preview>

```tsx title="button-basic.tsx" {3} showLineNumbers
import { Button } from "@/components/ui/button";

export default function Example() {
  return <Button>Click me</Button>;
}
```

</Preview>

## Variants

```tsx title="button-variants.tsx" {1,3,5} showLineNumbers
<Button variant="default">Default</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

## Sizes

```tsx title="button-sizes.tsx" {1,3,4} showLineNumbers
<Button size="sm">Small</Button>
<Button size="default">Default</Button>
<Button size="lg">Large</Button>
<Button size="icon" aria-label="Icon button">
  <span className="i-lucide-settings" />
</Button>
```
